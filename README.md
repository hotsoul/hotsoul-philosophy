### ビジョンの実現に向けて最初に目指すところは？
会社で農地を借り、福利厚生の一環として農作業を楽しみ、また収穫した作物を分け合って楽しめるような、働き方や社内制度、体制、新しい事業の仕組みをつくる。同時にそのような目標に共感する人を集め、ホットソウルの仲間になってもらう。
